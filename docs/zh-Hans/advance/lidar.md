## 激光雷达重建

激光雷达重建调用方法与基础重建一样，只是需要特定的数据输入，除了基础重建所需的参数以外，激光雷达需要设置或覆盖以下几个参数：


### 必需参数

| 字段名 | 类型 | 是否必需 | 说明 |
|--------|------|--------|------|
| `pipeline_mode` | int | 是 | 必须设置为1，表示纯激光雷达点云重建 |
| `coordinate_systems` | Array | 是 | 坐标系组  |
| `camera_meta_data` | JSON | 是 | 相机参数 |
| `image_meta_data` | JSON | 是 | 图像集合 |
| `lidar_data` | JSON | 是 | 激光雷达数据 |

详细参数介绍参考[激光雷达数据说明](./lidar-data)
