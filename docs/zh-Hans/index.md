## 概览

### 欢迎使用 MipMapEngine SDK

MipMapEngine SDK 是一个强大的三维重建工具包，专为航空摄影测量和三维建模而设计。无论您是初次接触三维重建技术，还是经验丰富的开发者，本文档都将帮助您快速上手并充分利用 SDK 的强大功能。

:::tip 
MipMapEngine SDK以CLI的形式提供，你可以使用C++，Python, JavaScript 等任何编程语言进行开发，或者在CMD, PowerShell等工具中调用执行。在评估使用SDK之前，强烈建议您先通过MipMapDesktop 软件评估模型重建相关的功能、性能和成果质量，完成评估后通过MipMapEngine评估集成到您的业务工作流中
:::

### SDK 能做什么？

MipMapEngine SDK 提供以下核心功能：

1. **空中三角测量**：全自动计算图像位置和姿态
2. **三维模型生成**：从图像生成高质量三维纹理网格模型
3. **正射影像/数字表面模型生成**：创建真正射影像和数字表面模型
4. **高斯泼溅生成**：生成逼真渲染的高斯泼溅成果
5. **实时重建**：边飞边出图，秒级实时获取二三维重建成果
6. **激光雷达重建**：输入激光雷达数据，生成三维纹理网格模型和高斯泼溅成果

### 系统要求

在开始之前，请确保您的系统满足以下要求：

- **操作系统**: Windows 10+ 或 Linux
- **显卡**: NVIDIA GPU (GTX 1050Ti 或更高)
- **驱动版本**: 
  - Windows: ≥ 528.33
  - Linux: ≥ 525.60.33
- **内存**: 16GB RAM (推荐 32GB+)
- **存储**: 至少 500GB 可用空间
